슬립(Slip): 아빠와 아들의 달리기
1. 비유를 통한 역할 매칭
- 아빠 (동기속도 Ns): 앞서 달려가는 회전자기장
- 아들 (실제속도 N): 아빠를 뒤따라가는 회전자(모터 축)
- 거리 차이 (슬립 s): 아빠보다 뒤처진 비율
2. 왜 아빠를 못 따라잡을까?
아들이 아빠와 똑같은 속도로 뛰면(s=0), 아빠가 아들을 끌어당기는 힘(유도 기전력)이 사라집니다. 따라서 모터가 계속 돌기 위해서는 반드시 아빠보다 조금 늦게 뛰어야만 합니다.
3. 상황별 슬립의 상태
- 정지 상태 (s=1): 아빠는 뛰기 시작했는데 아들은 아직 출발 안 함 (뒤처짐 100%)
- 운전 상태 (0 < s < 1): 아들이 아빠 뒤를 열심히 쫓아가는 중
- 동기 속도 (s=0): 아들이 아빠와 나란히 뜀 (힘이 안 생겨서 곧 다시 늦어짐)
4. 계산 포인트
아들의 속도(N) = 아빠 속도(Ns) × (1 - 슬립)
