유도전동기와 슬립 (Slip)
1. 유도전동기란?
고정자 코일에 전류를 흘려 만든 회전자기장이 회전자(모터 축)를 끌고 돌아가는 방식입니다.
- 별도의 전원 공급 없이 "유도"만으로 회전함
- 구조가 간단하고 튼튼해서 가장 많이 쓰임
2. 슬립 (Slip)의 정의
회전자기장의 속도(동기속도)보다 실제 모터가 얼마나 뒤쳐지는가를 나타내는 비율입니다.
s = (Ns - N) / Ns
- Ns: 동기속도 (자기장이 도는 속도)
- N: 실제속도 (모터 축이 도는 속도)
3. 슬립의 범위 (암기 필수!)
- 정지 시: s = 1 (100% 뒤쳐짐)
- 동기속도 회전 시: s = 0 (뒤쳐짐 없음)
- 정상 운전 시: 0 < s < 1
